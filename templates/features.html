All the features are listed here.

  1. User Accounts

    • Account Entry

      • Login

        • Users can log into their accounts.

      • Forgot Username

        • Users can receive an email with a list of their account usernames in case they forget their username.

      • Forgot Password

        • Users can receive an email link to reset the password for their account if they can’t remember it.

    • Account Creation

      • Signup

        • Users can sign up for an account if they do not have one. A confirmation email with the confirmation link for their account is sent to them.

      • Confirmation

        • Users can confirm their accounts so they can be activated for use.

    • Account Settings

      • Change Username

        • Users can change their username if they want to.

      • Change Password

        • Users can change their password if they feel the need to.

      • Change Email

        • Users can change their email if they change their email.

      • Delete Account

        • Users can delete their account if they don’t want it anymore. Note that this doesn’t exactly delete the account; it only revokes it’s confirmed status. The account confirmation link is sent to the user’s email if they ever want to confirm it again.

  2. Events

    • Event Creation

      • Create Event

        • Users can create an event.

    • Event Management

      • View All Events

        • Users can see all their events.

      • Edit Event Details

        • Users can modify the name and description for one of their events.

      • Delete An Event

        • Users can delete an event through the event editing page.

    • Event Finding

      • Searching For Events

        • Users can search for a specific event even if they are not logged in.

      • View An Event

        • Users can view a specific event even if they are not signed in.

  3. Forms

    • Form Creation

      • Create Form

        • Users can create a form for an event.

    • Form Management

      • View All Forms

        • Users can view all forms linked to an event.

      • Edit Form Details

        • Users can edit the details for a form.

      • Delete A Form

        • Users can delete a form through the form editing page.

      • Edit Form Questions

        • Users can create, edit, and delete questions for a form here.

      • View Form Answers

        • Users can view answers to a form.

      • Download Form Answers

        • Users can download answers to a form.

    • Form Finding

      • View All Forms For An Event

        • Users can look at all forms for an event.

      • View A Form

        • Users can view a specific form here.

      • Filling In A Form

        • Users can fill in a form and submit it here.

  4. Miscellaneous

    • Miscellaneous

      • Viewing The Homepage

        • Users can see the homepage.

      • Contacting the developers

        • Users can contact the developers and send their message or query.

      • Viewing The Timeline

        • Users can see the timeline.

      • Viewing The Features

        • Users can view the features of the website.
